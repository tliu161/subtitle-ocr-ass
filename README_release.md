# SubtitleOCR_ASS (Windows)

## How to use
1. Unzip `SubtitleOCR_ASS-windows.zip`
2. Open the folder `SubtitleOCR_ASS`
3. Double-click `SubtitleOCR_ASS.exe`
4. Click **Select Video**
5. Use the time slider to find a frame with subtitles
6. Drag to select the subtitle region (ROI)
7. Click **Run OCR â†’ ASS**
8. The output `.ass` file will be created next to your video file.

## Notes
- Keep the window open while OCR is running.
- First run may take longer (model files may be downloaded).
